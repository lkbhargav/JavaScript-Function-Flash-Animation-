Flash Animation for Webpage (Support for all platforms)


This is for people who want to use animations in their websites (for headings or any fancy stuff), usually we can use the .swf movie file that works great, but that is limited to desktops and laptops only, since flash is not supported on smartphones (iOS, Android, etc). That makes your website on smartphone look ugly with a missing plugin error, henceforth to overcome that problem, I have created a possible solution.

First of all, you should have a working flash movie of any length, at the same time it should be just animation, I mean to say it should'nt be interactive with buttons and music, since this tutorial is limited to plain animations. Once the flash movie is ready, you should convert the entire movie to a .jpg or .png sequence of images, you can get the sequence of images by exporting the movie in one of those formats. The main idea behind this is to change 24 images (any number) per second to get the animation effect. My work will make it easy for you to get the animations working real quick on your webpage. Anyways you should use the library or javascript file provided in this page to get it working.

Once you download the javascript file, make it available in you project directory along with the image sequence generated. Now include the javascript file into your project (specific html page), using the script tag and then follow the steps below.
